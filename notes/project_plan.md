# Project Plan

## Goal
Compare different neural network architectures for image classification.

## Datasets
- Fashion-MNIST
- CIFAR-10

## Models
- MLP (baseline)
- Simple CNN
- LeNet
- Improved CNN (custom)

## Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Visualizations
- Sample images
- Loss curves
- Accuracy curves
- Confusion matrices

## Notes
The goal is to understand how different architectures perform and why.
